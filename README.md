# Python酒店管理系统（源码+开发文档）简介

## 核心功能
- 房间管理
- 预订管理
- 顾客管理
- 收入管理
- 支出管理
- 员工管理
- 系统管理

## 适用场景
- 酒店行业
- 餐饮业
- 宾馆业
- 商务楼
- 企事业单位

## 技术特点
- 使用Python语言进行开发，基于Django框架
- 前端采用HTML、CSS、JavaScript等技术
- 后端使用MySQL数据库存储数据
- 支持Pycharm、navicat等开发工具
- 环境要求：Python3.0以上、mysql8.0
- 支持远程调试
